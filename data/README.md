# Data Directory

Place your dataset file here: `imdb_1000.csv`

## Dataset Format
- Columns: `review`, `sentiment`
- 1000 rows (500 positive, 500 negative)
- Source: IMDb Movie Reviews (Kaggle)

## Download
Get the dataset from: https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews

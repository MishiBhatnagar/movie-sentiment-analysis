## 📊 Project Overview

An end-to-end machine learning project that classifies IMDb movie reviews as **Positive** or **Negative** with **84% accuracy**.

| Metric | Value |
|--------|-------|
| **Best Accuracy** | 84.00% (SVM + TF-IDF) |
| **Dataset Size** | 1,000 IMDb reviews |
| **Models Compared** | 3 ML algorithms |
| **AUC Score** | 0.92 |
| **Test Samples** | 300 reviews |

### 📈 Model Performance

| Model | Accuracy | Precision | Recall | F1-Score |
|-------|----------|-----------|--------|----------|
| **Naive Bayes** | 79.67% | 79.87% | 79.33% | 79.60% |
| **SVM** | **84.00%** | **83.12%** | **85.33%** | **84.21%** |
| **Logistic Regression** | 79.67% | 76.97% | 84.67% | 80.63% |
